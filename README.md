# QANOUNI AI (v1.5)

**Smart Legal Consultant & Research Assistant for Algerian Law.**

## Overview
This repository contains the source code for QANOUNI AI, a RAG-based legal assistant.

## Tech Stack
- **Backend:** FastAPI, Python
- **Frontend:** Vanilla JS
- **Database:** Supabase
- **AI:** Groq (Llama 3), Gemini (Embeddings & Fallback)

## Deployment
Deployed on Vercel serverless functions.
